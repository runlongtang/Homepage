I wrote a simple tutorial (PDF) on how to use Git with a lot of screenshots. The tutorial just contains an example with details. For the basic idea of Git, you can read Pro Git by Scott Chacon and Ben Straub. Now I think Git is useful. I guess one with some knowledge about Linux and command lines can learn the basics in a couple of days. So, I strongly recommend people who use computers to write something learn to use Git. You don’t have to be a programmer to use Git. Both Git and Linux were created by Linus Torvalds, interesting. He really made some difference.
