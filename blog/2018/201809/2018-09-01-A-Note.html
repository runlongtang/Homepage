A Note on Big Data and AI Strategies by Kolanovic and Krishnamachari 2017

Runlong Tang

2018-09-01

 

The 2017 report Big Data and AI Strategies by Kolanovic and Krishnamachari provides a very good introduction to data collection and machine learning for Finance. Before reading this report, I did not know there were already so many data companies. I recommend anyone doing quantitative works on finance read this report.
